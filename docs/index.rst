Hand Tracking SDK
=================

Production-oriented Python SDK for consuming telemetry from Hand Tracking Streamer (HTS).

Key Features
------------

- Parse HTS wrist and landmarks telemetry into typed objects.
- Receive data over UDP, TCP server, or TCP client transports.
- Assemble coherent per-hand frames with sequence and timestamp metadata.
- Convert Unity left-handed geometry to right-handed coordinates.
- Stream events via high-level client APIs with filtering and observability.
